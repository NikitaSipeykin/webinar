
Коллекции в Java - это группы объектов, управляемые определенными структурами данных, которые могут содержать объекты
любых типов и имеют различные методы для работы с этими объектами. Коллекции предоставляют удобный способ хранения и
манипулирования данными в Java, и могут использоваться для решения широкого спектра задач, начиная от простых операций,
таких как добавление и удаление элементов, до более сложных, таких как сортировка и поиск. В Java существует множество
различных коллекций, включая списки, множества и карты, которые могут быть использованы в зависимости от конкретных
потребностей приложения.

8 примитивов - и каждый имеет свой класс обертку который привращает примитив в объект

Примитив    -   Класс обертка
int 	    -   Integer
short	    -   Short
long	    -   Long
byte        -	Byte
float       -	Float
double      -	Double
char        -	Character
boolean     -	Boolean
